## How to publish on GitHub Pages

1. Go to your repo on GitHub
2. Click **Settings**
3. Click **Pages** in the left sidebar
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://Praachee19.github.io/BLD/` within 2 minutes

## To connect a custom domain (e.g. bld.in)

1. In GitHub Pages settings, enter your domain under **Custom domain**
2. Go to your domain registrar (GoDaddy, Namecheap etc.)
3. Add these DNS A records pointing to GitHub:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
4. Wait 24 hours for DNS to propagate
